CS/A gecko parts list V1.0

These are the parts needed for the gecko system. 

	IC1		6502		  CPU
	IC2,8		74LS139		  dual 2-to-4 decoder
	IC3		74LS00		  4x NAND
	IC4		74LS06		  6x Inverter, Open Collector
	IC5		6264/62256	  8k or 32k static RAM
	IC6		2764/27128/27256  8k, 16k or 32k EPROM
	IC7		2 MHz Oscillator  
	IC9		74LS688		  8 bit compare
	IC10		16550A		  UART (with FIFO)
	IC11		6522		  VIA Versatile Interface Adapter
	IC12,13		MAX232		  RS232 level converter
	IC14		74LS14		  6x Inverter with Schmitt-Trigger
	IC15		74LS86		  4x XOR
	IC16		74LS245		  8 bit bidirectional bus driver
	IC17		74LS13		  2x NAND with Schmitt-Trigger
	IC18		74LS07		  6x Driver, Open Collector
	IC19		74LS145		  BCD to decimal decoder, O.C.

	R1		100k Ohm
	R2-5,7,9-13	4k7 Ohm
	R6		180 Ohm
	R8		33 Ohm
	R14		1k5 Ohm
	R15		1M Ohm
	R16		470 Ohm

	C1		100uF
	C2-7		47uF
	C8		100nF
	C9,10		10pF

	D1		1N4148
	D2,3		LD271 or LD274
	D4		LED, yellow

	Q1		Crystal 1.8432 MHz

	J1		3 pin		  RAM size 8/32 kByte
	J2		3 pin		  ROM size 8-16/32 kByte
	J3		3 pin		  I/O size 2/4 kByte
	J4		2 pin		  IEC ATNA enable

	P1		64 pin male DIN 41612 type C
	P2		9 pin D-Sub male (RS232)
	P3		CBM serial IEC female DIN 6 pin
	P4		2x5 pin
	P5		4 pin

	PB1		Reset pushbutton
	PB2-x		Keyboard buttons
